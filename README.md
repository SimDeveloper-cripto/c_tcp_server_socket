# C TCP Server Socket

This is an implementation of a tcp server socket, all written in C language.

## About folders

/bin contains the executable file.
/lib contains all header files.
/src contains all source files.

## How to test the server

### Compiling the code
```console
$ chmod +x build.sh
$ ./build.sh
```

### Execute the code
```console
$ cd bin
$ ./main
```
